      カンフーマンはこのゲームのサンプルキャラクターでありこのバージョン
　　　での動きを知るためのものです。

      カンフーマンを使うためにはselect.defの[Characters]の下に以下のように
　　　書いてください

      kfm

      これだけでカンフーマンは遊べます。

　　　カンフーマンの作業ファイルはエレクバイトで手に入れることができます
　　　そしてあなたはカンフーマンを変更・改変し配布する事ができます　　　　


  コントロール（動き）
  --------

    Basic Controls
    --------------
　　　カンフーマンは４ボタンのキャラです。
　　　Ｘボタンは弱いパンチ、Ｙボタンは強いパンチ
　　　Ａボタンは弱いキック、Ｂボタンは強いキック
　　　これからの説明で強いと言う意味をもつＳを強。
　　　弱いと言う意味のＷを弱とします

      D - 下
      F - 前
      U - 上
      B - 後
      x - Ｘボタン
      y - Ｙボタン
      a - Ａボタン
      b - Ｂボタン

　　　ＤＦのような続けざまに方向が書いてあったならそれは斜めのことです。
　　　Ａ＋ＢならばＡとＢを同時押しする事です。
　　　Ａ/ＢならばＡかＢを押す事です

　　　カンフーマンは２段ジャンプができます。
　　　空中で攻撃を出さない限りカンフーマンは空中でガードが可能です。
　　　
　　　カンフーマンは連続で技をくらわないように空中で体制を立て直すリカバー
　　　をするにはＸ＋Ｙです、その時に立て直したい方にキーを押していれば
　　　その方向に行きます。
　　　地面の近くでリカバーすると地面付近専用のリカバーをします。
      
　　　カンフーマンは挑発がスタートでできます。（１Ｐはenterキー）

    通常の動き
    -----------------
      U              - ジャンプ
      UF             - 前ジャンプ
      F              - 前移動
      DF             - 屈み
      D              - 屈み
      DB             - 屈みかガード
      B              - 後移動かガード
      UB             - 後ジャンプ
      F, F           - 前ダッシュ
      B, B           - 後ダッシュ（バックステップ）
      xy             - リカバー 

    通常立ち攻撃
    ----------------------
      X              - 弱パンチ
      Y              - 強パンチ
      A              - 弱キック
      B              - 強キック

    通常屈み攻撃
    -----------------------
      X              - 弱パンチ
      Y              - 強パンチ
      A              - 弱キック
      B              - 強キック

    通常空中攻撃
    ----------------------
      X              - 弱パンチ
      Y              - 強パンチ
      A              - 弱キック
      B              - 強キック

    投げ
    ------
      F/B + y 　　　 - カンフースロー

    必殺技
    ---------------
      D, DF, F + x   - 弱カンフー突き手
      D, DF, F + y   - 強カンフー突き手
      D, DF, F + xy  - 早いカンフー突き手

      F, F + a       - 弱カンフーニー
      F, F + b       - 強カンフーニー
      F, F + ab      - 早いカンフーニー

    超必殺技
    -------------
      D, DF, F, D, DF, F + x/y  - 三連カンフー突き手
