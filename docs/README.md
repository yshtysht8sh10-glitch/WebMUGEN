# Documentation policy

WebMUGEN documentation is maintained as standalone HTML files.

Rules:

- Each important document should support English and Japanese in the same HTML file.
- English is the default visible language.
- Use a small language switcher with `data-lang="en"` and `data-lang="ja"` sections.
- Keep documents readable in a browser without a build step.

日本語:

- WebMUGEN のドキュメントは単独 HTML ファイルで管理します。
- 重要なドキュメントは、同じ HTML 内で英語・日本語を切り替えられるようにします。
- 初期表示は英語にします。
- ビルドなしでブラウザから読めることを重視します。
