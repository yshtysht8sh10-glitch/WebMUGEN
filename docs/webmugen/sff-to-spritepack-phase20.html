SFF to SpritePack Phase20

This phase connects the SFF v1 parser and the PCX decoder.

  • SFF document to ImageDataSpritePack conversion
  • SFF sprite raw data to PCX decoding
  • PCX RGBA pixels to ImageData
  • Linked sprite resolution
  • ImageData sprite canvas cache helper

Phase21 will connect this ImageDataSpritePack to the main CanvasRenderer and character loading flow.
