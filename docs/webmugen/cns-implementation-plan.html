CNS Implementation Plan

CNSをWebMUGENに実装するための段階的計画。

実装順序

  1. CNS lexer/parserを作る。
  2. StateDefとState Controllerを内部形式へ変換する。
  3. trigger式を最低限評価する。
  4. ChangeState / VelSet / PosAdd / CtrlSet を動かす。
  5. HitDef最小実装で2P対戦を成立させる。
  6. AIR/SFFへ接続する。

最小対応trigger

  • time
  • animtime
  • command
  • ctrl
  • statetype
  • movehit

内部形式案

type StateController =
  | ChangeStateController

  | PosAddController
