CNS Metadata Tolerant Parser Phase25

This phase makes the CNS parser tolerant of real-world metadata sections used by KFM and other MUGEN characters.

  • Allows non-state sections such as [Data] and [Size]
  • Stores metadata sections separately
  • Ignores loose key-value pairs before any section
  • Keeps StateDef and State controller parsing behavior

This targets the previous KFM fallback error: Key-value pair appears outside a section.

CNS Metadata Tolerant Parser Phase25

このフェーズでは、KFMなど実キャラのCNSで使われるメタデータセクションを許容します。

  • [Data] や [Size] などState以外のセクションを許容
  • metadataSectionsとして別保存
  • セクション外のkey=valueは無視
  • StateDefとState Controllerの解析は維持

前回のKFM fallbackエラー Key-value pair appears outside a section の解消を狙います。
