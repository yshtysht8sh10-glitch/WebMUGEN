HitDef Phase8

Phase8では、固定ダメージの簡易攻撃から、CNSの HitDef Controllerに基づく攻撃へ移行する。

対応パラメータ

  • damage
  • pausetime
  • ground.velocity
  • air.velocity

テスト修正メモ

Phase7の古いテストでは「ヒットしたら必ず空中状態になる」と仮定していた。

Phase8では ground.velocity = -4, 0 のようにY速度が0なら、地上ヒットとして stateType = S のまま扱う。

相手を浮かせたい場合は、ground.velocity = -4, -4.5 のようにY速度を負にする。

現時点の仕様

  • HitDef が実行されると、攻撃側PlayerStateに activeHitDef を保持する。
  • 攻撃判定が接触すると activeHitDef のdamageを使う。
  • 1回当たったHitDefは hitDefUsed で再ヒットを防ぐ。
  • 本格的な hitflag、guardflag、attr、sparkno は未対応。
