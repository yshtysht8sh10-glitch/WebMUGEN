MUGEN-like Movement

MUGENのcommon stateを参考に、WebMUGENのサンプルCNSをよりMUGENらしい構造へ寄せる。

重要な変更

  • 歩きは State 20 に統合する。
  • State 20 内で command = "holdfwd" / command = "holdback" を毎フレーム見て VelSet する。
  • 斜めジャンプは、歩きより先に holdfwd_up / holdback_up を判定する。
  • 入力を離して State 0 に戻った時点で VelSet x = 0 する。

本物MUGENとの差分

本物MUGENでは State 40 で sysvar(1) にジャンプ方向を保存し、State 50 へ移る。現時点のWebMUGENは sysvar、ifelse、const が未実装のため、簡易的に 40 / 41 / 42 に分ける。

次に必要なもの

  • VelMul
  • VarSet / sysvar
  • const(...)
  • ifelse(...)
  • triggerall
