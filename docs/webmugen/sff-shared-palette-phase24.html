SFF Shared Palette Phase24

This phase adds shared palette support for SFF v1 sprites.

  • PCX decoder accepts an external 256-color palette
  • PCX decoder can detect embedded VGA palettes without throwing immediately
  • SFF converter scans sprites and finds a shared palette
  • Sprites without embedded palettes are decoded with the shared palette

This should move KFM loading past the previous error: PCX VGA palette marker is missing.

SFF Shared Palette Phase24

このフェーズでは、SFF v1 Spriteの共有パレットに対応します。

  • PCXデコーダが外部256色パレットを受け取れる
  • PCXデコーダが埋め込みVGAパレットを安全に検出できる
  • SFF変換処理がSprite群から共有パレットを探す
  • パレットを持たないSpriteを共有パレットでデコードする

これにより、KFM読み込み時の PCX VGA palette marker is missing. を越えることを狙います。
