PCX Decoder Phase19

This phase adds a minimal 8-bit indexed color PCX decoder for SFF v1 sprites.

Implemented

  • PCX header parser
  • 8-bit / 1-plane PCX validation
  • RLE decoding
  • 256-color VGA palette extraction
  • Indexed pixels to RGBA conversion
  • Palette index 0 transparency

Next

Phase20 will connect SFF sprite raw data to this PCX decoder and create Canvas-ready images.
