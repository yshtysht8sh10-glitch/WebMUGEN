Controller Executor Phase4

Phase4では、CNS State Controllerを実際にPlayerStateへ適用するExecutorを追加する。

対応Controller

  • ChangeState
  • VelSet
  • PosAdd
  • CtrlSet
  • ChangeAnim
  • StateTypeSet

重要な仕様

ChangeState が実行されたら、そのState内の残りControllerは実行しない。

Phase4では trigger は単純ANDとして評価する。MUGEN本来のtrigger1/trigger2規則は後続Phaseで対応する。

次のPhase

Phase5では、CnsDocumentのStateDefinitionとControllerExecutorを接続し、CNS定義に従ってキャラクターを動かす。
