Movement Phase6

Phase6では、CNS風テキストに基づく後退、ジャンプ、重力、着地を追加する。

修正メモ

ChangeState 直後は、遷移先Stateの time = 0 を次フレームで評価できるよう、stateTime を0のまま維持する。

これをしないと、遷移先Stateの VelSet trigger1 = time = 0 が一度も成立せず、移動しない。

追加した要素

  • command = "holdback"
  • command = "holdup"
  • Gravity Controller
  • pos y >= 285 trigger
