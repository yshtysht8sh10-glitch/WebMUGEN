App Character Loader Phase23

This phase connects CharacterLoader to the application entry point.

  • The app tries to load /chars/kfm/kfm.def.
  • If loading fails, the built-in sample character is used.
  • SFF-derived ImageDataSpritePack is passed to CanvasRenderer.
  • PNG SpritePack fallback remains available.
