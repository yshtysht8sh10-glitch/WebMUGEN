Hit State Phase9

Phase9では、HitDefが当たった相手を簡易被弾Stateへ遷移させる。

追加State

  • 5000: 簡易立ち食らい
  • 5030: 簡易空中食らい

対応

  • 地上ヒット時に stateNo = 5000
  • 浮かせヒット時に stateNo = 5030
  • moveType = H
  • ctrl = false
  • 一定時間後に State 0 へ復帰
  • VelMul Controllerの最小対応

本物MUGENとの差分

本物MUGENでは、5000番台にはHitShakeOver、GetHitVar、HitOver、HitVelSetなど多くの専用処理がある。Phase9では、まず「被弾状態に入る」ことだけを目的とする。

次のPhase

Phase10では、AIR読み込みへ進み、animNoだけでなく実際のアニメーション定義に基づいて描画する。
