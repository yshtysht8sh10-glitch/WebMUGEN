MUGEN-like AnimTime Trigger Phase46

This phase makes AnimTime work like a remaining animation time trigger.

  • Adds animation duration helper
  • Calculates MUGEN-like AnimTime as remaining frames
  • Passes AIR animation duration into CNS runtime from the app
  • Evaluates AnimTime = 0 using remaining time, not elapsed time
  • Supports KFM State 200 returning via AnimTime = 0
