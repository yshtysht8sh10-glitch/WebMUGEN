WebMUGEN Architecture

長期開発とMUGEN互換性を見据えた基本設計。

レイヤ構造

WinMUGEN / MUGEN 1.0 / MUGEN 1.1 assets
        ↓
parser + compatibility layer
        ↓
WebMUGEN internal definitions
        ↓
core engine
        ↓
renderer / audio / input / netplay

重要な分離

  • coreはReact、Canvas、PHPを知らない。
  • rendererはGameStateを描画するだけ。
  • version差はmugen配下とcompatibility layerで吸収する。
