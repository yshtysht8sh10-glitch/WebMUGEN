CMD Phase13

Phase13では、MUGENのCMDファイルの最小パーサとCommandResolverを追加する。

対応

  • [Command] ブロック
  • name
  • command
  • time
  • /F, /B, /U, /D, a
  • F+U, B+U の簡易斜め入力

目的

今までCNSの command = "a" などはKeyboardInputSourceが直接解決していた。Phase13以降はCMD定義を経由してcommand名を解決する。

未対応

  • 波動拳入力などの時系列コマンド
  • ~, $, +, / の完全なMUGEN互換
  • 入力バッファ
