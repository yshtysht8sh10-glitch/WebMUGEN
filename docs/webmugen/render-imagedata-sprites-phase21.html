Render ImageData Sprites Phase21

This phase connects ImageDataSpritePack to the main CanvasRenderer.

  • CanvasRenderer accepts ImageDataSpritePack
  • ImageData sprites are converted to cached offscreen canvases
  • AIR group/image can resolve to SFF-derived ImageData sprites
  • PNG SpritePack remains as fallback
  • Fallback stick-figure drawing still works when no sprite is found

Phase22 will add a character asset loader that fetches def/air/cns/cmd/sff files.

Render ImageData Sprites Phase21

このフェーズでは、ImageDataSpritePackをメインCanvasRendererへ接続します。

  • CanvasRendererがImageDataSpritePackを受け取れる
  • ImageData Spriteをキャッシュ済みの内部Canvasへ変換
  • AIRのgroup/imageからSFF由来ImageData Spriteを解決
  • PNG SpritePackはフォールバックとして残す
  • Spriteが見つからない場合は従来の棒人間描画を継続

Phase22では、def/air/cns/cmd/sffを取得するキャラクターアセットローダーを追加します。
