WebMUGEN Documentation

WebMUGEN is a browser-based MUGEN-like fighting game engine prototype.

Implemented phases

  • CNS parser and state machine
  • AIR parser and animation player
  • CMD parser and input buffer
  • Clsn collision boxes
  • HitDef and hit states
  • Projectile controller
  • SpritePack renderer prototype
