WebMUGEN Docs

This is the main documentation index for WebMUGEN. English is the default because it is easier for most coding assistants and external contributors to parse consistently.

Purpose

WebMUGEN aims to run MUGEN-style characters in the browser by parsing DEF, CNS, AIR, CMD, and eventually SFF/SND files.

Current engine areas

  • CNS state machine and controllers
  • AIR animation and Clsn boxes
  • CMD command matching and input buffer
  • HitDef, hit states, Projectile
  • SpritePack renderer prototype before full SFF support

Recommended reading order

  2. CNS parser and StateMachine
  3. AIR / Clsn
  4. CMD / InputBuffer
  5. Projectile
  6. SpritePack and future SFF loader

目的

WebMUGEN は、DEF、CNS、AIR、CMD、将来的には SFF/SND を読み、MUGEN 風キャラクターをブラウザ上で動かすことを目指しています。

現在のエンジン領域

  • CNS ステートマシンと Controller
  • AIR アニメーションと Clsn 判定
  • CMD コマンド判定と入力バッファ
  • HitDef、被弾 State、Projectile
  • 本格 SFF 対応前の SpritePack レンダラー
