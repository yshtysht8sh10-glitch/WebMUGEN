Character Loader Phase22

This phase adds the first character asset loader.

  • DEF parser
  • [Files] extraction
  • cns / air / cmd loading
  • sff loading and conversion to ImageDataSpritePack
  • fetcher abstraction for tests and future zip support

Phase23 will connect this loader to the application flow and add a KFM loading experiment.
