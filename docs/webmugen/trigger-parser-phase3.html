Trigger Parser Phase3

Phase3では、CNSのtrigger式を最小ASTへ変換し、実行時に評価できる土台を追加する。

修正メモ

0 と 1 はbooleanではなく数値として扱う。MUGENの式では time = 0 のような比較が頻出するため。

statetype = A のような右辺の A は、未定義識別子ではなく文字列リテラル相当として扱う。

対応例

  • time = 0
  • time > 10
  • animtime = 0
  • command = "holdfwd"
  • command != "holdback"
  • ctrl
  • statetype = A
  • movehit
