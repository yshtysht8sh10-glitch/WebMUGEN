WinMUGEN CMD Notes

CMD files define named commands from input sequences.

Example

[Command]
name = "qcf_a"
command = D, DF, F, a
time = 20

Current behavior

WebMUGEN stores recent inputs in an input buffer and resolves active command names each frame.
