WinMUGEN Spec Notes

This area records researched WinMUGEN behavior so future work does not need to rediscover the same details.

Files

  • .def: character metadata and file references
  • .cns: states, triggers, and controllers
  • .air: animations and Clsn boxes
  • .cmd: command definitions
  • .sff: sprite archive
  • .snd: sound archive

Documentation policy

Each spec note should include both English and Japanese sections in one standalone HTML file. English should be shown by default.

ここには、WinMUGEN の仕様調査メモを残します。後で Codex や別の生成AIへ引き継いだときに、同じ仕様を調べ直さないための場所です。

ファイル

  • .cmd: コマンド定義

  • .snd: サウンドアーカイブ
