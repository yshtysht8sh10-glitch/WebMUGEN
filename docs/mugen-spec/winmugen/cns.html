WinMUGEN CNS Notes

CNS files define character state logic.

StateDef

[StateDef 200]
type = S
movetype = A
physics = S

State Controllers

Controllers such as ChangeState, VelSet, HitDef, and Projectile modify player state during the game loop.

WebMUGEN scope

WebMUGEN currently parses states, triggers, common controllers, hit definitions, movement, hit states, and projectile creation.
