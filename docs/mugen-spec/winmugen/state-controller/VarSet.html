VarSet

Variable / WinMUGEN CNS State Controller調査メモ

位置づけ

この文書はWebMUGEN実装用の初期メモであり、最終的なWinMUGEN完全仕様ではない。実機検証・既存キャラ検証・MUGEN 1.0差分調査を通じて更新する。

概要

キャラクター変数を設定する。

代表パラメータ

パラメータ意味・調査観点
v整数変数番号。
fv浮動小数変数番号。
value設定値。

記述例

[State 1000, Set]
type = VarSet
trigger1 = time = 0
v = 0
value = 1

WebMUGEN実装方針

PlayerState.variablesとしてint/floatを保持する。
