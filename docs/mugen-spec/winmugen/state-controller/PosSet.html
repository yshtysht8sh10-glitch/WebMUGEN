Movement / WinMUGEN CNS State Controller調査メモ

位置づけ

この文書はWebMUGEN実装用の初期メモであり、最終的なWinMUGEN完全仕様ではない。実機検証・既存キャラ検証・MUGEN 1.0差分調査を通じて更新する。

概要

位置を直接設定する。

代表パラメータ

パラメータ意味・調査観点
xX座標。
yY座標。

記述例

[State 0, ResetPos]
type = PosSet
trigger1 = time = 0
y = 0

WebMUGEN実装方針

座標系の原点、地面位置、カメラとの関係を分けて設計する。
