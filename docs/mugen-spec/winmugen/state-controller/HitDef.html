HitDef

Combat / WinMUGEN CNS State Controller調査メモ

位置づけ

この文書はWebMUGEN実装用の初期メモであり、最終的なWinMUGEN完全仕様ではない。実機検証・既存キャラ検証・MUGEN 1.0差分調査を通じて更新する。

概要

攻撃判定とヒット時挙動を定義する。

代表パラメータ

パラメータ意味・調査観点
attr攻撃属性。
damageダメージ。
hitflag当たる相手状態。
guardflagガード可否。
pausetimeヒットストップ。
ground.velocity地上ヒット時速度。

記述例

[State 200, Hit]
type = HitDef
trigger1 = time = 0
attr = S, NA
damage = 30
hitflag = MAF
guardflag = MA
pausetime = 12,12
ground.velocity = -4,-2

WebMUGEN実装方針

WebMUGEN互換の中核。最初はdamage、hitbox接触、hitstun、knockbackだけに絞る。
