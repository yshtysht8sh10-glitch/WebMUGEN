CtrlSet

Control / WinMUGEN CNS State Controller調査メモ

位置づけ

この文書はWebMUGEN実装用の初期メモであり、最終的なWinMUGEN完全仕様ではない。実機検証・既存キャラ検証・MUGEN 1.0差分調査を通じて更新する。

概要

プレイヤー操作可否を変更する。

代表パラメータ

パラメータ意味・調査観点
value1で操作可、0で操作不可。

記述例

[State 200, Ctrl]
type = CtrlSet
trigger1 = animtime = 0
value = 1

WebMUGEN実装方針

入力処理そのものではなく、State遷移条件側でctrlを参照する。
